# Array Algorithms in Java

A beginner-friendly collection of essential array algorithms implemented in Java. This repository covers searching techniques and common array operations that form the foundation of Data Structures and Algorithms (DSA).

## 📚 Programs Included

### 🔍 Searching Algorithms
- Linear Search
- Binary Search

### 📊 Array Operations
- Find Maximum Element in an Array
- Find Minimum Element in an Array
- Calculate Average of Array Elements
- Count Occurrences of an Element
- Find Second Largest Element
- Reverse an Array

## 🎯 Purpose

This repository was created to practice and understand:
- Array traversal techniques
- Searching algorithms
- Problem-solving using arrays
- Time complexity analysis
- Basic DSA concepts in Java

## ⏱️ Time Complexity

| Algorithm | Time Complexity |
|-----------|----------------|
| Linear Search | O(n) |
| Binary Search | O(log n) |
| Find Maximum Element | O(n) |
| Find Minimum Element | O(n) |
| Calculate Average | O(n) |
| Count Occurrences | O(n) |
| Find Second Largest | O(n) |
| Reverse Array | O(n) |
